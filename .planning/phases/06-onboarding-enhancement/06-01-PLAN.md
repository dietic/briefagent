Step 1 - Type selection (shown first):
- Header: `m.onb_quick_type_title()` + subtitle `m.onb_quick_type_subtitle()`
- Three clickable cards in a vertical stack (similar to the existing card styling: `rounded-[14px]`, `border`, CSS vars):
