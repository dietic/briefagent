**POST endpoint (`src/routes/api/social-accounts/+server.ts`):**

Create a POST handler that:
1. Gets authenticated user via `locals.getUser()` -- return 401 if not authenticated
2. Parses JSON body: `{ productId: string, platform: string, handle?: string, url?: string }`
3. Validates: `productId` and `platform` are required, at least one of `handle` or `url` must be provided
4. Verifies ownership: query `products` table to confirm `product.userId === user.id` -- return 403 if not owned
5. Inserts into `socialAccounts` table using Drizzle: `db.insert(socialAccounts).values({ productId, platform, handle, url }).returning()`
6. Returns 201 with the created social account JSON

Use `import type { RequestEvent } from '@sveltejs/kit'` (not `$types`) for handler typing -- consistent with Phase 4 pattern.

**DELETE endpoint (`src/routes/api/social-accounts/[id]/+server.ts`):**

Create a DELETE handler that:
1. Gets authenticated user via `locals.getUser()` -- return 401 if not authenticated
2. Gets social account ID from `params.id`
3. Loads the social account with its product relation to verify ownership: query `socialAccounts` table, join with `products` to check `product.userId === user.id` -- return 403/404 if not found or not owned
4. Deletes the social account: `db.delete(socialAccounts).where(eq(socialAccounts.id, id))`
5. Returns 200 with `{ success: true }`

For the ownership check, use a pattern consistent with existing API endpoints:
```ts
const account = await db.query.socialAccounts.findFirst({
  where: eq(socialAccounts.id, id),
  with: { product: true }
});
if (!account || account.product.userId !== user.id) return json({ error: 'Not found' }, { status: 404 });
```

**Settings page server (`src/routes/dashboard/settings/+page.server.ts`):**

Load social account counts per product to show in settings cards. After loading `allProducts`:
```ts
import { socialAccounts } from '$lib/server/db/schema';
import { count } from 'drizzle-orm';

// For each product, get the count of social accounts
const productIds = allProducts.map(p => p.id);
const accountCounts: Record<string, number> = {};
for (const pid of productIds) {
  const result = await db.select({ count: count() }).from(socialAccounts).where(eq(socialAccounts.productId, pid));
  accountCounts[pid] = result[0]?.count ?? 0;
}
```

Return `socialAccountCounts: accountCounts` alongside `settingsProducts`.
