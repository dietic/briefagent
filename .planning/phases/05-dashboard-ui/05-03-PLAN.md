**Radial chart component** (`src/lib/components/dashboard/radial-chart.svelte`):

Two modes via props: `mode: 'score' | 'donut'`

Score mode props: `{ mode: 'score'; value: number; max?: number; color: string; size?: number }`
- Single arc SVG showing value/max as a circular progress
- SVG circle with `stroke-dasharray` and `stroke-dashoffset` for the arc
- Background circle: stroke --border-subtle, 3px wide
- Foreground arc: stroke color (passed prop), 4px wide, stroke-linecap round
- Center text: value as large font-extrabold number, "/100" below in text-xs --text-muted
- Size default 120px
- CSS transition on stroke-dashoffset for animation on mount (use `$effect` to trigger)

Donut mode props: `{ mode: 'donut'; segments: { percentage: number; color: string; label: string }[]; size?: number }`
- Multiple arcs, each segment starting where the previous ended
- Use `stroke-dasharray` with gap segments
- Center text: total percentage (should be ~100%) or a label
- Size default 160px
- Legend rendered below the SVG: colored dots + label + percentage
